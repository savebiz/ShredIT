# Project Plan: DataGuard Green Shredding Platform

> **Generated by:** @project-planner
> **Date:** February 2026
> **Focus:** Full Stack Web Platform (MVP + Landing Page)

## 1. Executive Summary
This plan outlines the implementation of the **DataGuard Green Shredding Platform**, a SaaS solution transforming corporate paper waste into verified carbon credits.
**Strategy:**
1.  **Phase 1 (Validation)**: Public Landing Page to validate value prop (using `/ui-ux-pro-max`).
2.  **Phase 2 (MVP)**: Core Platform (User Mgmt, Dashboard, Collection Requests) using Next.js + Supabase.
3.  **Phase 3 (Growth)**: Green Points & Carbon Tracking (Future Scope).

## 2. Technical Architecture
**Stack:**
-   **Framework**: Next.js 14 (App Router)
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS (with `ui-ux-pro-max` design system)
-   **Database**: PostgreSQL (via Supabase)
-   **Auth**: Supabase Auth (Email/Password + Magic Link)
-   **State**: React Hooks / Context (Zustand if needed)
-   **Icons**: Lucide React

**Data Model (Supabase):**
-   `tenants` (companies)
-   `users` (role-based: admin, manager, branch_user)
-   `sites` (locations)
-   `bins` (assets)
-   `collections` (transactions)
